## Array (Ruby dynamic array)
method      | time complexity
------------|----------------
push        | O(1)
pop         | O(1)
shift       | O(1)
unshift     | O(n)
[] (index)  | O(1)
find        | O(n)
insert      | O(1)

## LinkedList
 methods                 | time complexity
-------------------------|-------
push, shift              | O(1)
pop, unshift             | O(1)
find                     | O(n)
indexing([])             | O(n)
remove_at/insert         | O(n)

## Queue
methods                 | time complexity
------------------------|-------
enqueue                 | O(1)
dequeue                 | O(1)
max/min                 | O(n)
peek                    | O(1)

## Stack
methods                 | time complexity
------------------------|-------
push                    | O(1)
pop            | O(1)
max/min                    | O(1)
peek                       | O(1)
## StackQueue
methods                 | time complexity
------------------------|-------
enqueue             | O(1)
dequeue         | O(1) amortized
max/min                    | O(1)
peek                       | O(1) amortized
